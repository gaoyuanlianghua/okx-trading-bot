# OKX交易机器人

OKX交易机器人是一个基于12层架构设计的加密货币交易自动化工具，支持OKX交易所的API操作和WebSocket实时数据获取。

## 功能特点

- 支持OKX交易所的REST API和WebSocket API
- 实现了内外核双循环架构，兼具实时响应和深度认知能力
- 提供直观的GUI界面，方便用户操作和监控
- 支持多种交易策略，包括动态策略和PassivBot策略
- 内置风险控制和异常处理机制
- 支持代理配置，应对网络限制

## 系统架构

项目采用12层架构设计，结合内外核双循环机制：

- **外循环**：实时感知-行动环，负责快速响应和即时交互
- **内循环**：认知学习-进化环，负责深度认知和长期进化

## 安装与配置

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置API密钥

在运行前，需要在`main.py`中配置OKX API密钥：

```python
api_key = "你的API密钥"
secret_key = "你的API密钥密码"
passphrase = "你的API密钥密码短语"
```

### 3. 运行程序

```bash
python main.py
```

## 使用方法

1. **启动程序**：运行`main.py`文件，启动交易机器人
2. **配置交易参数**：在GUI界面中设置交易对、方向、数量等参数
3. **手动交易**：点击"买入"或"卖出"按钮进行手动交易
4. **启动策略**：选择策略类型，点击"启动策略"按钮启动自动化交易
5. **监控状态**：在各个标签页中查看市场数据、账户余额、订单状态等信息

## 策略说明

### 动态策略

动态策略是一种基于随机决策的交易策略，会在设定的时间间隔内随机决定买入或卖出操作。

### PassivBot策略

PassivBot策略是一种基于网格交易的自动化策略，通过在不同价格水平设置买卖订单来获取市场波动收益。

## 注意事项

1. **风险提示**：加密货币交易存在高风险，请谨慎使用本工具
2. **API密钥安全**：请妥善保管你的API密钥，不要分享给他人
3. **网络环境**：如果遇到网络连接问题，可以尝试配置代理
4. **测试建议**：在使用实盘交易前，建议先在模拟环境中测试策略

## 故障排除

- **API连接失败**：检查API密钥是否正确，网络连接是否正常
- **WebSocket连接断开**：系统会自动尝试重连，无需手动操作
- **订单执行失败**：检查账户余额是否充足，交易对是否正确
- **GUI界面无响应**：可能是网络请求阻塞，等待一段时间后会自动恢复
